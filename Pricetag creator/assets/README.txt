Приложение написано на языке python 3.9

Функционал основан на работе с Excell таблицами и ручным вводом товарных данных по заданому шаблону

Кнопка Вывод из Excell читает последний ввод из буфера обмена и продолжает цикл по заданому числу повторений

Кнопка Просмотр файла открывает cardsFile.txt файл из директории программы для чтения и модификации вручную

Кнопка Вывод этикеток запускает браузер по умолчанию. Для отрисовки границ и форматирования текста ценников, для этого браузер извлекает информацию из файла cardsFile.txt

Кнопка Ручной ввод читает набранный текст в соответствующем окне для и записывает в файл для последующего вывода.

Кнопка Перезапись удаляет всю прежде полученную информацию из файла cardsFile.txt для новой сессии записи

Кнопка Локальный сервер запускает сервер на локальном компьютере для отрисовки браузером графики
